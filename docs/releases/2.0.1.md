# MKT KSA Geolocation Security v2.0.1

Security-focused release for production Rust services and smart-city trust workflows.

![Release Cover](https://raw.githubusercontent.com/mktmansour/MKT-KSA-Geolocation-Security/main/docs/images/mkt_ksa.png)

## Release Highlights

- Adaptive AI guard hardening with burst-aware risk handling.
- Deterministic JWT-first authorization semantics.
- Dynamic Retry-After behavior for rate-limit responses.
- Strict request correlation and trace continuity (`X-Request-ID`).
- Environment-tunable HTTP runtime hardening controls.
- Bilingual technical documentation and structured wiki rollout.

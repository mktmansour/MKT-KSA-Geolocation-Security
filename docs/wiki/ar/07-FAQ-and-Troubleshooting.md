# 07. الأسئلة الشائعة وحل المشاكل

تجمع هذه الصفحة أكثر الأسئلة تكرارًا في التشغيل والتكامل.

## 1. لماذا تظهر استجابات unauthorized؟

تحقق من:

- قيمة API key واسم الرأس الصحيح.
- صلاحية JWT ومصدر التوقيع.
- فرق الوقت بين مصدر التوكن والخادم.

## 2. لماذا يظهر rate limit بسرعة؟

تحقق من:

- نمط الحركة وحجم الاندفاع.
- إعدادات RATE_LIMIT_MAX_REQUESTS.
- وجود عدة عملاء خلف نفس IP.

## 3. لماذا يتم الحظر بسبب المخاطر؟

تحقق من:

- وجود مؤشرات مشبوهة في body.
- تطابق الحمولة مع قواعد المنع.
- حساسية endpoint وفق الملف الأمني.

## 4. لماذا يفشل الإقلاع في بيئة جديدة؟

تحقق من:

- غياب API_KEY أو JWT_SECRET.
- خطأ في DATABASE_URL.
- عدم وجود صلاحية كتابة لمسار SQLite.
- عدم توافق أداة Rust مع lockfile.

## 5. كيف أجهز النشر إلى crates.io بأمان؟

قائمة تحقق:

- مراجعة include داخل Cargo.
- فحص ملفات الحزمة قبل النشر.
- استبعاد ملفات الاختبار والتشغيل من payload.

## 6. أوامر مفيدة

```bash
cargo check --locked
cargo test --all
cargo package --allow-dirty --list
cargo publish --dry-run --allow-dirty
```
